## How to Use

### Getting Started
1. Open `index.html` in a web browser
2. The system will automatically load with all menu items

### Making a Sale
1. Click on products to add them to the cart
2. Use +/- buttons to adjust quantities
3. Click "Complete Order" to process the sale
4. View/print the receipt
5. Click "New Order" to start fresh

## Technical Details

### Technologies Used
- **HTML5**: Structure and layout
- **CSS3**: Modern, responsive styling
- **JavaScript**: All functionality and logic
- **LocalStorage**: Data persistence (products and sales)

### Data Storage
- All data is stored locally in your browser's LocalStorage
- Products and sales persist between sessions
- No internet connection required after initial load

### Browser Compatibility
- Works on all modern browsers (Chrome, Firefox, Safari, Edge)
- Optimized for tablets (iPad, Android tablets)
- Responsive design works on phones and desktops too

### Customization
You can easily customize:
- Color scheme (edit `styles.css`)
- Product categories
- Receipt format
- Tax calculations (currently set to 0%)

## Installation

### For Tablet Use
1. Copy all files (index.html, styles.css, script.js) to your device
2. Open `index.html` in a browser
3. Add to home screen for app-like experience:
   - **iOS**: Safari > Share > Add to Home Screen
   - **Android**: Chrome > Menu > Add to Home Screen

### Logo Setup
- Replace `logo.png` with your actual MugShots Cafe logo
- Recommended size: 500x500 pixels
- Transparent background (PNG format) works best

## Tips for Best Experience

1. **Use a tablet in landscape mode** for the best POS layout
2. **Add to home screen** for quick access
3. **Regular backups**: Export your sales data periodically
4. **Update product images**: Add real photos of menu items for better customer experience
5. **Training**: Familiarize staff with all features before going live
